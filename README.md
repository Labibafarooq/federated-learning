# Federated Learning 

This repository contains exercises and experiments related to Distributed and Federated Learning.

The goal of these experiments is to understand how machine learning models can be trained collaboratively across multiple clients without sharing raw data.

## Topics Covered

- Federated Learning concepts
- Client-server training architecture
- Distributed model updates
- Privacy-preserving training
- Aggregation of model parameters

## Technologies Used

- Python
- PyTorch
- Jupyter Notebooks
- Federated Learning concepts

## Repository Structure

exercise1 – Introduction to distributed training  
exercise2 – Model training with multiple clients  
exercise3 – Aggregation of client updates  
exercise4 – Performance comparison in federated settings  
exercise5 – Experiments with decentralized learning strategies

## Purpose

These exercises were completed as part of coursework on Distributed and Federated Learning and aim to provide practical understanding of collaborative machine learning systems.
