# Federated Learning

This repository contains coursework and experiments related to Distributed and Federated Learning.

## Topics covered
- Distributed training
- Federated learning workflows
- Multi-client experiments
- Performance evaluation with metrics such as accuracy, precision, recall, F1-score, and loss
- Experiment folders containing plots and results

## Structure
The repository is organized by exercises, and each exercise may contain additional subfolders for code, data, experiment settings, and result figures.

## Tools
- Python
- PyTorch
- Flower
- Jupyter Notebook
